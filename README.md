# LLM-ассистированный перевод — GitHub Pages

## Публикация
1) Создайте репозиторий (например: llm-icc-module).
2) Загрузите содержимое этой папки в корень репозитория.
3) Settings → Pages → Source: Deploy from a branch → Branch: main → /(root)
4) Откройте опубликованную ссылку.

## Материалы
Файлы шаблонов кладите в downloads/ и обновляйте ссылки в downloads.html.
